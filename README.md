## Linear Regression
the idea is to fit a straight line to the data.
```math
y = mx+b
```
where, y = predicted value, x = input variable, m = slope, b = intercept

linear regression finds the line that minimizes the difference between actual values and predicted values
this is done using a method called least squares, which minimizes the sum of squared errors.

## Logistic Regression
